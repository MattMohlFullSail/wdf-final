h1
This is the most heirarchically important heading tag. It is a block-level tag.
h2
This is simply one level lower than the 'h1' tag. The only difference being the text is slightly smaller by default.
h3
Same as h2, just the text is slightly smaller than h2.
p
The paragraph element. Contains simple text that has no other classification besides copy.
ul
Unordered list. A block-level tag that contains list items. The list items will have a bullet to the left of them by default.
ol
Ordered list. Block-level that also contains list items, however, they will have a number next to them that increase incrementally.
li
List item. Signifies an item in a list. These appear inside of an ol or ul block-level element.
